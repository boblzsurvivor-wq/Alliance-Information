## Overview

This guide explains why diplomats and alliance help are so powerful, using simple math.

The goal is to show how small time reductions translate into large diamond savings, and how that scales across an alliance.

---

## 💎 Diamond Value per Player

**Rule:**  
1 hour = 100 diamonds

If diplomats provide **+10 hours of help per project**:

- Each project saves **1,000 diamonds**
- 100 projects = **100,000 diamonds saved per player**

This is the baseline value from consistently using alliance help.

---

## 🕒 Project Length vs Value

As long as the full help is applied:

- **10 hours = 1,000 diamonds saved (always)**

What changes is the percentage impact:

- 50h project → **20% faster**
- 100h project → **10% faster**
- 200h project → **5% faster**
- 400h project → **2.5% faster**

**Key points:**

- Diamond savings are constant  
- Time saved is constant  
- Percentage impact decreases as projects get longer  

Longer projects do not reduce value — they just make the effect less visible.

---

## 🔁 Efficient Use of Saved Diamonds

From 100 projects:

- **100,000 diamonds saved**
- Equivalent to **1,000 hours of speedups**

These diamonds can be used more effectively:

- Applied as speedups when needed  
- Saved for events where diamonds have higher return  
- Used on priority upgrades instead of timers  

**Key point:**  
Diplomats reduce waste and improve overall efficiency.

---

## 👥 Alliance Scale (100 Players)

If 100 players each complete 100 projects:

- Total projects = **10,000**

Total savings:

- **10,000,000 diamonds**
- **100,000 hours**

This is the impact when everyone contributes consistently.

---

## ‼️ Why Everyone Matters

The system scales with participation.

If players do not help:

- Less time is reduced  
- Fewer diamonds are saved  
- Efficiency drops across the alliance  
- Progress slows overall  

Even small gaps in participation lead to large losses at scale.

---

## 📊 Summary

- 100 projects = **100,000 diamonds saved per player**  
- Savings are fixed per project once help is used  
- Longer projects reduce visibility, not value  
- Saved diamonds can be used more efficiently elsewhere  
- 100 players = **10,000,000 diamonds saved**  

**Small actions repeated across the alliance create massive gains**
